**Plataforma de comercio electrónico para la venta y distribución de productos artesanales 100% mexicanos.**

Este proyecto busca conectar a los artesanos locales con un mercado global, ofreciendo una interfaz intuitiva, segura y representativa de la cultura nacional.

##  Instalación y Ejecución

Para visualizar el proyecto en local:

1.  Clonar el repositorio:
    ```bash
    git clone [https://github.com/tu-usuario/xmex-platform.git](https://github.com/tu-usuario/xmex-platform.git)
    ```
2.  Navegar a la carpeta del proyecto:
    ```bash
    cd xmex-platform
    ```
3.  Abrir el archivo `src/index.html` en tu navegador web de preferencia.
