# Introducción
El presente proyecto consiste en el desarrollo de una plataforma web orientada a la comercialización de artesanías mexicanas, creada y operada completamente en México por comerciantes mexicanos. La finalidad del sistema es brindar un espacio digital donde los artesanos y comerciantes puedan publicar sus productos, mientras que los clientes pueden visualizar, seleccionar y realizar pedidos de manera sencilla y segura.

Este proyecto se desarrolla como parte de la materia Fundamentos de Ingeniería de Software, aplicando conceptos básicos como análisis de requerimientos, diseño de base de datos y documentación del sistema. La plataforma busca simular un entorno real de comercio electrónico, promoviendo el consumo local y la preservación de la identidad cultural mexicana mediante el uso de artesanías tradicionales.

Alcance del Proyecto

El alcance del proyecto incluye el diseño y documentación de un sistema web que permita:

El registro e inicio de sesión de usuarios.

La diferenciación de usuarios entre clientes, comerciantes y administrador.

La publicación, edición y eliminación de productos artesanales por parte de los comerciantes.

La visualización de un catálogo de productos con imágenes y descripciones.

La creación y consulta de pedidos asociados a los usuarios.

El diseño de una base de datos que gestione usuarios, productos y pedidos.

El proyecto no incluye la implementación de pagos en línea reales, sistemas de envío ni validaciones legales, ya que estos quedan fuera del alcance académico y se consideran posibles mejoras futuras.

Objetivos del Proyecto
Objetivo General

Diseñar y documentar una plataforma web de venta de artesanías mexicanas que permita la gestión de usuarios, productos y pedidos, aplicando los fundamentos de la ingeniería de software.

Objetivos Específicos

Analizar los requerimientos funcionales y no funcionales del sistema.

Diseñar una base de datos relacional que almacene la información del sistema.

Crear contenido visual y descriptivo para simular un sistema funcional.

Documentar el sistema de manera clara y estructurada.

Promover el comercio local y la cultura mexicana mediante una solución tecnológica.
